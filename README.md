# ReviewInsight - 评论分析工具

一个基于React的评论分析工具，支持上传CSV、Excel和JSON文件进行AI驱动的评论分析。

## 功能特点

- 文件上传支持（CSV、Excel、JSON）
- 拖放上传界面
- 表单验证
- 响应式设计
- 美观的UI界面

## 技术栈

- React 18
- Vite
- react-dropzone
- CSS3

## 开始使用

### 安装依赖

```bash
npm install
```

### 开发服务器

```bash
npm run dev
```

### 构建项目

```bash
npm run build
```

## 部署到Vercel

1. 确保您已经安装了Vercel CLI：
   ```bash
   npm i -g vercel
   ```

2. 登录到Vercel：
   ```bash
   vercel login
   ```

3. 部署项目：
   ```bash
   vercel --prod
   ```

4. 按照提示完成部署过程

部署完成后，您的应用将可以通过Vercel提供的URL访问。
